# 第三阶段录屏演示脚本

目标时长：2-3 分钟。

1. 打开主界面，停留在平静星空，说明这是“把情绪折成星星”的 Demo。
2. 点击“记录情绪”，输入一段委屈文字，选择“委屈”，点击完成。
3. 展示原始信纸和纸团流程：点击“折成纸团”，再点击纸团投向星空。
4. 投掷结束后停留 1-2 秒，展示委屈角色、狐狸、玫瑰和雨滴恢复物件。
5. 点击雨滴，强调未点够前只显示“轻轻点亮雨滴 2/5”这类进度，不提前显示完成。
6. 点够数量后展示提示：“这颗星星已经被你安放好了。”
7. 再快速注入或记录 2-3 条同类情绪，展示主星空出现淡淡星座线和治愈提示。
8. 打开“观测星空”，展示统计、情绪筛选、日期筛选和更明显的星座连线。
9. 点击一颗星打开详情，展示原文、情绪、日期、星座名、收藏和删除。
10. 打开“改变环境”，展示同一情绪下的雨夜星空、炉边星空、海浪星空、摇篮星空四张场景变体。
11. 选择一个环境场景，说明它只切换星空陪伴氛围并推荐音频，不会自动播放。
12. 点击播放白噪音，演示雨声/篝火/海浪/摇篮曲与音量调节。
13. 最后简短展示“手势实验”入口，说明它是可选增强，正式演示仍使用鼠标。

结束语：第三阶段的核心价值来自完整主流程、可点击恢复仪式、逐渐成形的星座、可回看的观测星空，以及稳定可控的环境面板。

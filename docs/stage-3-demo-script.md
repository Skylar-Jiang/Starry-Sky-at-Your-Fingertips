# 第三阶段录屏演示脚本

目标时长：2-3 分钟。

1. 打开主界面，停留在平静场景，说明这是“把情绪折成星星”的 Demo。
2. 点击“记录情绪”，输入一段委屈文字，选择“委屈”，点击完成。
3. 展示信纸变成待折纸条，点击“折成纸团”，再点击纸团投向星空。
4. 投掷结束后停留 3 秒，展示委屈角色、同伴靠近、玫瑰低垂和细雨反馈。
5. 点击场景中的雨滴互动点，让雨滴逐个消散，清完后展示“场景恢复平静”。
6. 点击“观测星空”，展示统计面板，选择“非常难过”或“委屈”筛选。
7. 展示星座连线和星座装饰，点击一颗星打开星星详情。
8. 在详情里展示日期、星座名、原始文字，点击收藏，再关闭或删除演示记录。
9. 打开“改变环境”，选择雨声，点击播放白噪音，调节音量。
10. 再记录一条“焦虑”，投掷后展示沙尘泡泡互动。
11. 简短说明手势实验入口是可选增强，默认不影响键鼠流程。

结束语：第三阶段的核心价值来自可回看的星空、可操作的环境、可点击的恢复反馈，以及稳定可录屏的演示流程。

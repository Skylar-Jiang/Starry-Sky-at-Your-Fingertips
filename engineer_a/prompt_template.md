# 第一阶段 AI 生图提示词模板

## 背景图

用途：`assets/background/bg_calm.png`、`bg_happy.png`、`bg_sad.png`

要求：
- 1920x1080。
- PNG 或 JPG。
- 治愈、童话、星空、水彩、柔和。
- 不要文字、不要水印、不要明显版权角色。
- 需要留出按钮和弹窗可读区域。

## 透明底角色与物件

用途：主角、狐狸、玫瑰、纸张、纸团、星星、雨滴、光点。

要求：
- PNG 透明底。
- 不要文字、不要水印、不要白底。
- 不要原版小王子形象，保持“童话旅人”气质即可。
- 主角建议 1024x1024。
- 狐狸、玫瑰、星星、纸团建议 512x512 或 1024x1024。

## 命名规则

严格使用 `asset_checklist.md` 中的文件名。技术组按同名文件接入，不改代码路径。
